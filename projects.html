Temperature Converter for Android

Similar to the Temperature Converter for iOS this app has mostly the same functionality (my app is pictured above). It takes an input from the user, this time in the form of a textbox, and the user has the option to select from two buttons. One converts the input from Celcius to Fahrenheit, while the other button does Fahrenheit to Celcius. Contrary to the iOS app, the Android version does not use a slider for temperature input. File Explorer

One of my favorite projects I have done during my time at St. Thomas was a File Explorer in my Operating Systems course. We were given a graphics library to use in order to fully implement a working file explorer on our linux systems. In addition to showing file sizes, types, and names, it also allowed for the user to click a button an enable recursive mode. This way the user could see multiple subdirectories without needing to click so many times. It was a rather difficult project but it is very cool to say you made your own file explorer.

Electoral College Simulator

In Object Oriented Design and Programming, one of my favorite projects was the Electoral College Simulator. Its purpose was pretty self-explanatory but it essentially took a list of Red, Blue, and Undecided States, and out-put how many ways you could end up with a tie of electoral college votes. It was a difficult project but it was cool to implement because it used fork/join and allowed for a direct performance comparison to the method without fork/join.

Tenzi Dice Game

The last project on this page is another from Object Oriented Design and Programming. Tenzi Dice Game was one of the very first 'games' I ever developed and it is also one of my favorite. The purpose of the game was to see how long it took for the player to have all dice land on the same number. For instance, if the user had 3 dice then they would keep rolling until all dice had the same number, but could select how many of the 3 dice to roll each turn. The program kept track of the elapsed time and kept a leaderboard of the fastest recent times.
